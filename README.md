# Playground JS

Мини-проект для практики: кнопки, DOM, console.log, эксперимент с let/const.

## Что сделано

- index.html, script.js, style.css
- Кнопка Alert
- Кнопка изменения заголовка
- Примеры использования const и let

## Как запустить

1. Открыть `index.html` через Live Server в VS Code
2. Или: открыть файл в браузере
